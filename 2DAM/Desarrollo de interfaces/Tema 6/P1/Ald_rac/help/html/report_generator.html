Generador de informes

En esta sección se podrá generar informes PDF utilizando archivos jrxml

Generar un informe

.- Sitúe los informes jrxml en la carpeta "resources"" que encontrará en la ruta donde se haya instalado el programa.

.- Rellene los campos proporcionados

.- Se recomienda probar la conexión a la base de datos pulsando en "Test" antes de generar el informe

.- Si la conexión a la base de datos es correcta, pulse en "Generate" para generar el informe.

Los informes generados se guardarán automáticamente en la carpeta "reports" donde esté instalado el programa.
