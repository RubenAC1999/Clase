Tipos de < input >
Tipo Descripción Ejemplo
text Entrada estándar de texto
password Igual que una entrada estándar de texto pero está oculto
submit Envía el formulario a la página que está indicada en el atributo action
button Crea un botón que se puede configurar con JavaScript
hidden Sirve para ocultar texto
image Pone una imagen que nos envía al formulario
file Indica un archivo que enviar con el formulario
checkbox Pone una caja en la que podemos marcar o desmarcar Opción 1 Opción 2 Opción 3
radiobutton Pone una caja en la que solo podemos marcar una opción Opción 1 Opción 2 Opción 3
email Campo para email
tel Campo para teléfono
url Campo para url
number Un campo para números
range Creamos una barra de rango
search Campo para búsqueda
color Creamos un pequeño selector de color
date Campo para una fecha
time Campo para una hora
datetime-local Campo para indicar fecha y hora
week Campo para indicar una semana
month Campo para indicar el mes
